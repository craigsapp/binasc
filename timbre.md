These examples demonstrates how to choose an instrument for playing notes on a MIDI channel.  The MIDI command for selecting timbres is C0`-`CF hex, where the top nibble is always `C` which indicates a [patch change](http://en.wikipedia.org/wiki/Synthesizer#Patch), and the bottom nibble is a channel number for 0 (for channel 1) to `F` (for channel 16).

Each MIDI channel can have only one active timbre.  Since MIDI allows for up to 16 channels on a single device, this means that up to 16 timbres can be played simultaneously.  Note that MIDI file tracks are independent of the channel: you can use multiple channels within a single track (such as a Type-0 MIDI file), or you can separate each channel into a separate track, or separate tracks can share the same channel (such as violins I & II).

  * [Example 1](#Example_1.md): Playing a single note in each General MIDI timbre.
  * [Example 2](#Example_2.md): Mozart double canon K228/KV515b played with all General MIDI timbres in succession.
  * [Example 3](#Example_3.md): Playing a single note in each percussion timbre.

# Example 3 #

---


This example MIDI file performs one note for each of the 47 percussion timbres available on channel 10.  Percussion timbres are chosen by playing keyboard notes on General MIDI channel 10 rather than by selecting a timbre with the `C0` MIDI patch-change command.  This is because unpitched percussion instruments have no pitch, so 127 notes to represent the full range of the instruments is not necessary.
